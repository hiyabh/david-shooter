# 🚀 David Shooter — משחק חלל

משחק יריות בחלל בסגנון קלאסי, בנוי עם HTML5 Canvas וג'אווהסקריפט טהור.

## 🎮 איך משחקים

פתח את `game.html` בדפדפן, או שחק אונליין:
👉 **[לחץ כאן לשחק](https://hiyabh.github.io/david-shooter/game.html)**

### בחירת דמות
בחר בין שני שחקנים — כל אחד עם חללית בצבע משלו.

### שליטה
| מקש | פעולה |
|-----|--------|
| ↑↓←→ / WASD | הזזת החללית |
| רווח | ירי |
| X | טיל על-קולי (מפוצץ הכל!) |
| R | חזרה לבחירת דמות |

## ✨ תכונות

- מסך בחירת דמות עם תצוגה מקדימה
- חללית עם פרצוף בחלון קוקפיט
- כוכבים מסתובבים עם פיצוצי חלקיקים
- טיל על-קולי עם גל הלם שמנקה את המסך
- קושי עולה לפי שלבים
- בונוס טיל כל 200 ניקוד

## 🛠️ טכנולוגיות

- HTML5 Canvas
- JavaScript (Vanilla)
- ללא תלויות חיצוניות
